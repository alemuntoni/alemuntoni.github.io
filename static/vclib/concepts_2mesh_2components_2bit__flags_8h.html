 HasBitFlags concept is satisfied only if a Element class provides the member functions specified in this concept. These member functions allows to access to a BitFlag component of a given element.

concept  vcl::comp::HasTriangleBitFlags
 HasTriangleBitFlags concept is satisfied only if a Element class (that should be a Face) provides the member functions specified in this concept. These member functions allows to access to TriangleBitFlags component of a given element.
 
concept  vcl::comp::HasFaceBitFlags
 HasFaceBitFlags concept is satisfied if one between HasPolygonBitFlags and HasTriangleBitFlags concept is satisfied. This concept allows to make sure that a Face element has proper FaceBitFlags (Triangle or Polygon).
