virtual vcl::BitSet< short > vcl::proc::SaveMeshAction::supportedInputMeshType ( ) const
inlinevirtual

Returns a BitSet that tells, for each mesh type, if the action supports it or not.

By default, all mesh types are supported.

You should override this method if your action does not support all mesh types.

Returns
A BitSet with the supported mesh types.
