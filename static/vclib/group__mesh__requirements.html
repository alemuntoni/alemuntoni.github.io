◆ isPerElementComponentEnabled()

template<uint ELEMENT_TYPE, uint COMPONENT_TYPE, MeshConcept MeshType>
bool vcl::isPerElementComponentEnabled ( const MeshType &  m)

Returns true if the given component is available (enabled) in the given element of the input mesh m.

This function returns true when the component can be used on the element, whether the component is horizontal, vertical or optional.

These are the following cases:

  • if the Mesh does not have an ElementContainer of the given ElementEnumType, the function returns false;
  • if the Element does not have a Component of the given ComponentEnumType, the function returns false;
  • if the Element has a non-optional Component of the given ComponentEnumType, the function returns true;
  • if the Element has an optional Component of the given ComponentEnumType, the function returns true if the component is enabled, false otherwise (this check is the only one that is made at runtime);
Template Parameters
ELEMENT_TYPEthe ElementEnumType of the Element to check.
COMPONENT_TYPEthe ComponentEnumType of the Component to check.
MeshTypethe type of the Mesh to check.
Parameters
[in]mthe mesh on which check the availability of the Component in the Element.
Returns
true if the Component is available in the Element of the given Mesh.

◆ requireElementContainerCompactness()

template<uint ELEMENT_TYPE, MeshConcept MeshType>
void vcl::requireElementContainerCompactness ( const MeshType &  m)

This function asserts that a Mesh has the Container of the given ElementEnumType compact (no elements flagged as deleted).

If the Container of the given ElementEnumType is not compact, a vcl::MissingCompactnessException will be thrown. If the mesh has not a Container of the given ElementEnumType, a build error will be raised.

Template Parameters
ELEMENT_TYPEthe ElementEnumType of the ElementContainer to check.
MeshTypethe type of the Mesh to check.
Exceptions
vcl::MissingCompactnessExceptionif the Container of the given ElementEnumType is not compact.
Parameters
[in]mthe mesh on which check the compactness of the Container of the given Element.

◆ requirePerElementComponent()

template<uint ELEMENT_TYPE, uint COMPONENT_TYPE, MeshConcept MeshType>
void vcl::requirePerElementComponent ( const MeshType &  m)

This function asserts that a Mesh has the Container of the given ElementEnumType, the Element has a Component of the given ComponentEnumType, and that Component is enabled and available at runtime.

If the Mesh:

  • has not a Container of the given ElementEnumType;
  • has the Container but the Element has not a Component of the given ComponentEnumType;
  • has the Container, the Element has the Component, but the Component is not enabled a vcl::MissingComponentException will be thrown.
Template Parameters
ELEMENT_TYPEthe ElementEnumType of the Element to check.
COMPONENT_TYPEthe ComponentEnumType of the Component to check.
MeshTypethe type of the Mesh to check.
Exceptions
vcl::MissingComponentExceptionif the Component is not enabled or available in the Element Container of the Mesh.
Parameters
[in]mthe mesh on which check the availability of the Component in the Element.
