Point#

template<typename Scalar, int N>
class Point#

The Point class represents an N-dimensional point containing N scalar values.

Subclassed by vcl::Point2< Scalar >, vcl::Point3< Scalar >, vcl::Point3< float >, vcl::Point4< uint8_t >
