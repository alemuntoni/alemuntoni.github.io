Detailed Description

template<uint EL_TYPE, typename MeshType>
struct vcl::mesh::ContainerOfElement< EL_TYPE, MeshType >

The ContainerOfElement structure exposes the type of the container of the given MeshType having the given ElementEnumType EL_TYPE.

If no container was found, the usage of this structure will cause a build error.

Template Parameters
EL_TYPEThe ElementEnumType of the Element
MeshTypeThe MeshType of the Mesh from which we need the type of the Container of Element having the given ElementEnumType
