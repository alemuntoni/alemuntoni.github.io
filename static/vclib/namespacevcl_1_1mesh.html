template<uint ELEM_ID, typename MeshType >
using vcl::mesh::ContainerOfElementType = typedef typename ContainerOfElement<ELEM_ID, MeshType>::type

The ContainerOfElementType structure exposes the type of the container of the given MeshType having Element with ID ELEM_ID.

If no container was found, the usage of this structure will cause a build error.

Template Parameters
ELEM_IDThe ElementEnumType of the Element
MeshTypeThe MeshType of the Mesh from which we need the type of the Container of Element having the given ELEM_ID

Usage:

using VertexContainer = mesh::ContainerOfElementType<ElemId::VERTEX, MyMesh>;
