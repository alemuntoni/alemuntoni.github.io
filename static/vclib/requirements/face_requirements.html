template<typename MeshType>
bool isFaceContainerCompact(const MeshType &m)

Returns true if the given mesh has its Face Container compact. Returns false if the mesh has no Face Container.

Parameters

m – [in] input mesh on which test if the its Face Container is compact.

Returns

true if the Face Container of the mesh is compact.
