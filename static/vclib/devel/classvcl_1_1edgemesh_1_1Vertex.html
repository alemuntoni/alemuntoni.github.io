Detailed Description

template<typename Scalar, bool I>
class vcl::edgemesh::Vertex< Scalar, I >

The Vertex type used by the EdgeMeshT class.

Template Parameters
ScalarThe scalar type used for the mesh.
IA boolean flag that indicates whether the mesh uses indices or pointers to store vertices of edges and adjacency information.
