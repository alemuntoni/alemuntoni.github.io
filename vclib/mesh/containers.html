Element ContainersÔÉÅ

Element Containers in VCLib are classes having the goal to contain a collection of Elements of the same type. These Containers are then inherited by a Mesh class that is composed (between the others) of a given Container of Elements.

Here is the list of Containers provided by VCLib:

  • Vertex Container
  • Face Container
  • Edge Container
