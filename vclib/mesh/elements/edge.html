The Edge Class#

The Edge Class is one of the possible Elements of a Mesh of VCLib. Itâ€™s duty is just contain a set of Components or Optional Components, exposing the member functions of each component.

All the components that can be part of Edges are defined inside the vcl::edge namespace, but you always can include any other component in your custom Edge definition.

When it is included in a Mesh definition, the Mesh will automatically instantiate a Edge Container, exposing its members to the Mesh class.
