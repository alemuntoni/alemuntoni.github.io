virtual MeshParamVector vcl::proc::FilterMeshAction::inputMeshParameters ( ) const

Returns a vector that describes the input meshes required by the action.

The vector contains pairs of MeshParameter and BitSet<short>. The MeshParameter describes the input mesh, while the BitSet<short> tells which mesh types are supported for that input mesh.

Returns
