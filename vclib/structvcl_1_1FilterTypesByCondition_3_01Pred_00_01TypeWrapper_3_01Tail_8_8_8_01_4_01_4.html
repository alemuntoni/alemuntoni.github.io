Detailed Description

template<template< class > class Pred, typename... Tail>
struct vcl::FilterTypesByCondition< Pred, TypeWrapper< Tail... > >

Removes all types that do not satisfy a condition, and get them as a tuple.

The condition is a templated predicate struct that must have a static member bool value that is true if the type satisfies the condition.

The result is the type type alias, that is a TypeWrapper containing the types that satisfy the condition.

The input types can be a pack of types or a TypeWrapper.

Usage:

using ResTypes =
FilterTypesByCondition<std::is_integral, int, float, double, char>::type;
static_assert(std::is_same<ResTypes, TypeWrapper<int, char>>::value, "");
